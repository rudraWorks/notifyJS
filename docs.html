The notify() function take a maximum of 9 arguments. Title, Message, Direction, Type, bgColor, textColor, btnColor, btnText and opacity are the attributes.



  1. Title

    notify('i am title');


  2. Message

    notify('i am title','this is message');


  3. Direction

    notify('i came from left direction','this is message','left');

    notify('i came from right direction','this is message','right');

    notify('i came from top direction','this is message','top');

    notify('i came from bottom direction','this is message','bottom');

    the default value of direction is top

  4. type

    notify('title','','','message');

    notify('title','','','danger');

    notify('title','','','alert');

    notify('title','','','success');

    notify('title','','','');

    the default value of type is '' ; that is empty

  5. bgColor

    notify('title','green','left','message','grey');

    notify('title','green','left','message','pink');

    notify('title','orange','left','message','orange');

    notify('title','default','left','message','');

    the default value of bgColor is white

  6. textColor

    notify('title','text text text text text','left','message','tomato','blue');

    the default value of textColor is white

  7. btnColor

    notify('title','text text','left','success','','blue','red');

    the default value of btnColor is skyblue

  8. btnText

    notify('title','text text','left','success','','blue','red','exit');

    the default value of btnText is 'ok'

  9. opacity

    notify('title','text text','left','success','pink','blue','red','exit','.5');

    the default value of opacity is 1

To use default value of a parameter, just use '' (empty quotes).
For example to just change the alert box bgcolor, use   notify('','','','','red','','','','');  

Installation: Place src="https://rudraworks.github.io/notifyJS/notify.js"
in script tag after the DOM loads(bottom of body tag).
